# Privacy Policy — Daily Question Journal

**Last updated: February 8, 2026**

---

## English

### Overview

Daily Question Journal is a personal journaling app that stores all data exclusively on your device. No data is collected, transmitted, or shared with any third party.

### Data Storage

- All journal entries and settings are stored locally on your device using an SQLite database.
- No data is sent to any server, cloud service, or third party.
- No account creation is required.

### Data Collection

This app does **not** collect:

- Personal information beyond the first name you optionally provide (stored only on your device)
- Usage analytics or tracking data
- Location data
- Advertising identifiers
- Any data transmitted over the internet

### Permissions

- **Notifications**: The app may request permission to send local notifications for daily reminders. These notifications are scheduled entirely on your device and do not involve any server communication.
- **File System Access**: Used solely for the export/import backup feature, allowing you to save and restore your journal data as a local file.

### Export & Import

The app allows you to export your journal data as a JSON file and share it using your device's sharing menu. This file remains under your control at all times. The app also allows importing data from a previously exported backup file. No data is uploaded to any server during this process.

### Third-Party Services

This app uses **no** third-party services, SDKs, analytics tools, advertising frameworks, or crash reporting services.

### Data Deletion

You can delete all your data at any time by using the "Reset app" function in the Settings screen, or by uninstalling the app.

### Children's Privacy

This app does not knowingly collect any data from children. Since no data is collected or transmitted at all, the app is safe for users of all ages.

### Changes to This Policy

Any changes to this privacy policy will be reflected in app updates. The "Last updated" date at the top will be revised accordingly.

## Deutsch

### Überblick

Daily Question Journal ist eine persönliche Tagebuch-App, die alle Daten ausschließlich auf Ihrem Gerät speichert. Es werden keine Daten erhoben, übertragen oder an Dritte weitergegeben.

### Datenspeicherung

- Alle Tagebucheinträge und Einstellungen werden lokal auf Ihrem Gerät in einer SQLite-Datenbank gespeichert.
- Es werden keine Daten an Server, Cloud-Dienste oder Dritte gesendet.
- Es ist keine Kontoerstellung erforderlich.

### Datenerhebung

Diese App erhebt **keine**:

- Persönlichen Informationen über den optional eingegebenen Vornamen hinaus (wird nur auf Ihrem Gerät gespeichert)
- Nutzungsanalysen oder Tracking-Daten
- Standortdaten
- Werbe-Identifikatoren
- Über das Internet übertragene Daten jeglicher Art

### Berechtigungen

- **Benachrichtigungen**: Die App kann um Erlaubnis bitten, lokale Benachrichtigungen für tägliche Erinnerungen zu senden. Diese Benachrichtigungen werden vollständig auf dem Gerät geplant und beinhalten keine Serverkommunikation.
- **Dateisystemzugriff**: Wird ausschließlich für die Export-/Import-Backup-Funktion verwendet, die es Ihnen ermöglicht, Ihre Tagebuchdaten als lokale Datei zu sichern und wiederherzustellen.

### Export & Import

Die App ermöglicht den Export Ihrer Tagebuchdaten als JSON-Datei und das Teilen über das Freigabemenü Ihres Geräts. Diese Datei bleibt jederzeit unter Ihrer Kontrolle. Die App ermöglicht auch den Import von Daten aus einer zuvor exportierten Sicherungsdatei. Während dieses Vorgangs werden keine Daten auf einen Server hochgeladen.

### Drittanbieterdienste

Diese App verwendet **keine** Drittanbieterdienste, SDKs, Analysetools, Werbeframeworks oder Absturzberichtserstattungsdienste.

### Datenlöschung

Sie können jederzeit alle Ihre Daten löschen, indem Sie die Funktion „App zurücksetzen" im Einstellungsbildschirm verwenden oder die App deinstallieren.

### Datenschutz für Kinder

Diese App erhebt wissentlich keine Daten von Kindern. Da keinerlei Daten erhoben oder übertragen werden, ist die App für Nutzer jeden Alters sicher.

### Änderungen dieser Richtlinie

Änderungen dieser Datenschutzrichtlinie werden in App-Updates berücksichtigt. Das Datum „Zuletzt aktualisiert" wird entsprechend angepasst.
